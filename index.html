📱 아이폰에서 앱처럼 쓰는 방법 (App Store 설치 없이)

1. 이 HTML 파일을 iCloud Drive(파일 앱)에 저장하거나, GitHub Pages 등에 올려 링크로 접속하세요.
2. Safari로 이 페이지를 여세요.
3. 하단 공유 버튼(⬆️) → "홈 화면에 추가" 를 누르면 앱 아이콘처럼 홈 화면에 생기고, 이후에는 아이콘을 눌러 앱처럼 실행할 수 있습니다. (App Store 설치가 아니라 웹페이지 바로가기입니다.)
4. 인터넷 연결 없이도 번호 생성 자체는 동작합니다 (설정값은 이 기기에 저장되어 있으므로).

⚠️ 복권은 매 회차 완전 독립적인 무작위 추첨입니다. Hot/Cold 번호를 사용해도 당첨 확률은 수학적으로 동일합니다 — 이 도구는 재미를 위한 것입니다.
